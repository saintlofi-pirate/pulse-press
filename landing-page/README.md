# Moonfarmer Reactions Lead Capture Landing Page

Static marketing page for Moonfarmer Reactions Lead Capture.

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8080 --directory landing-page
```

Then visit:

```text
http://localhost:8080
```

## GitHub Pages

This folder is self-contained and can be published from GitHub Pages.

Recommended setup:

1. Push this repository to GitHub.
2. Go to repository **Settings > Pages**.
3. Choose the branch to publish.
4. If GitHub Pages does not support publishing directly from `landing-page/`, copy this folder to the publishing branch root or configure a GitHub Actions Pages workflow.

The page uses only static HTML, CSS, and local assets.
